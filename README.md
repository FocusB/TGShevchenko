# TGShevchenko

Мені тринадцятий минало.
Я пас ягнята за селом.
Чи то так сонечко сіяло,
Чи так мені чого було?
Мені так любо, любо стало,

Прийшла, привітала,
Утирала мої сльози
І поцілувала .....
Неначе сонце засіяло,
Неначе все на світі стало
Моє... лани, гаї, сади!..
І ми, жартуючи, погнали
Чужі ягнята до води.
