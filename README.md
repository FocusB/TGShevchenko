# TGShevchenko

Мені тринадцятий минало.
Я пас ягнята за селом.
Чи то так сонечко сіяло,
Чи так мені чого було?
Мені так любо, любо стало,

Неначе в бога ......
Уже прокликали до паю,
А я собі у бур'яні
Молюся богу... І не знаю,
Чого маленькому мені
Нема в мене хати!
