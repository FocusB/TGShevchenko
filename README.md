# TGShevchenko

Мені тринадцятий минало.
Я пас ягнята за селом.
Чи то так сонечко сіяло,
Чи так мені чого було?
Мені так любо, любо стало,
19-26
Неначе в Бога......
І рай запалило.
Мов прокинувся, дивлюся:
Село почорніло,
Боже небо голубеє
І те помарніло.
Поглянув я на ягнята —
Не мої ягнята!
